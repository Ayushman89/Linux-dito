# Linux-dito
linux dito helps you to create a linux environment on a pc or smartphone 
using a code editor
## tested in
`` Mobile, Windows ``
## note
it is not a complete linux environment provider 
some basic commands you can use are ``pwd`` ``ls`` ``cd``
